# disaster-crawler

A web crawler that can gather and summarize information about the Noto Peninsula earthquake damages from the Ishikawa prefecture's website. The program should collect data on human casualties, damages to buildings and infrastructure, and any localized effects due to the earthquake. The site's disaster management page is identified as a useful resource.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/disaster-crawler.git
cd disaster-crawler
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
